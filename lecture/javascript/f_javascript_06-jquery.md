# javascript plugin (jQuery)

지난시간엔 반복문에 관한 내용에 대해 알아보았습니다.
이번시간에는 예외 처리라는 부분과 기타 내용들에 대한 부분을 이해해 보도록 하겠습니다.

---

 프로그램의 구조는 2가지로 구분이 가능합니다.

- 기술된 순서대로 처리를 행하는 **순차** 구조
- 조건에 따라 처리를 분기하는 **선택** 구조
- 특정의 처리를 반복 실행하는 **반복** 구조
